## Overview
This project demonstrates the implementation of Continuous Integration and Continuous Delivery (CI/CD) for a machine learning model using GitHub Actions. The system automates building, testing, and validating an ML-powered API.

## Features
- Machine Learning model served via API
- Containerization using Docker
- Automated CI/CD pipeline using GitHub Actions
- Automated testing using pytest

## Prerequisites
- Python 3.8+
- pip

## Installation
pip install -r requirements.txt

## Run the Application
python app/main.py

## Run Tests
pytest tests/

## CI/CD Pipeline
The GitHub Actions pipeline automatically:
- Installs dependencies
- Starts the application
- Runs automated tests
- Validates the build before deployment
